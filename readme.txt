# McDonald's Menu Nutrition Analysis

This repository contains the files and scripts for the McDonald's Nutrition Analysis project.  
The goal of this project is to analyse nutritional data for McDonald's menu items and investigate differences in calories, fat, and other nutrients between sandwich types.

## Project Structure
- `dataset/` - CSV dataset file
- `scripts/` - R scripts for data cleaning, EDA, and statistical analysis
- `reports/` - Report template for the assignment
